# OPDbot — репетиторский Telegram-бот для расписания

[![Python](https://img.shields.io/badge/Python-3.10+-blue.svg)](https://python.org)
[![Telegram](https://img.shields.io/badge/Telegram-Bot-blue.svg)](https://core.telegram.org/bots)

## 🎯 О проекте

Telegram-бот для автоматизации напоминаний о занятиях в репетиторском центре. 
Бот парсит Excel-файл с расписанием (несколько листов: расписание, ученики, преподаватели) 
и отправляет уведомления ученикам и преподавателям за 20 и 19 минут до урока.

### Возможности
- ✅ Регистрация по ролям (`ученик` / `преподаватель`)
- ✅ Привязка к ID ученика (число в квадратных скобках)
- ✅ Автоматическое определение часовых поясов (мск, мск+2, мск+5 и т.д.)
- ✅ Команды: `/start`, `/my_lessons` (или кнопка "Показать расписание")
- ✅ Фоновая проверка занятий (раз в 30 секунд)
- ✅ Поддержка нескольких напоминаний (20 и 19 минут)

## 🛠 Технологии
- `pyTelegramBotAPI` — взаимодействие с Telegram
- `openpyxl` — парсинг Excel-файлов
- `pytz` — работа с часовыми поясами
- `threading` — фоновый поток для напоминаний
